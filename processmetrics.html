Process Metrics

Story Completion Rate:
The number of tasks or user stories that are currently in progress and not completed yet.
We will track how many stories are completed each sprint vs how many were assigned and adjust from there.

Project Metrics

Defect Tracking:
Bug Rate: Number of bugs existing in each block of code.
We will track our bugs as stories in Jira and assign each defect to a specific team member to fix.

App Performance:
Response Time: It measures how long the app will take to respond to the user interactions.
We will track and measure the response time of the application to the user interations. If the response time takes too long, we will optimize it.

User Experience:
Ease of Use: How long it takes users to perform common tasks in the app.
We will ensure that the application's instructions are clear and straightforward, allowing users to complete tasks efficiently.
